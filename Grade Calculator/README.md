# GRADE CALCULATOR

A simple and responsive web application that helps users calculate their grades easily. The project is built using HTML, CSS, JavaScript, and Bootstrap to provide a clean interface and accurate grade calculation.

Overview:

The Grade Calculator allows students or users to input their marks and instantly calculate their grade results. It is designed with a responsive layout so it works smoothly on desktop and mobile devices.

This project demonstrates basic front-end development concepts such as form handling, JavaScript logic, responsive design, and user-friendly interface design.

Features:

1.Easy grade calculation from user input

2.Clean and responsive user interface

3.Built using Bootstrap for modern layout

4.Simple and lightweight code structure

5.Fast calculations using JavaScript

6.Beginner-friendly project structure

Technologies Used:

-->HTML5 – Structure of the application

-->CSS3 – Styling and layout design

-->JavaScript – Grade calculation logic

-->Bootstrap – Responsive UI components
